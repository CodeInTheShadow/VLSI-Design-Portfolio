## 🛠️ Repository Structure
The project is organized into modular directories based on logic complexity:

1. **01_Combinational_Logic:** Basic and advanced arithmetic circuits, including an **8-bit ALU** and 4-bit Ripple Carry Adders.
2. **02_Sequential_Logic:** Implementation of various Flip-Flops (JK, SR, T) and complex counters (4-bit Johnson, Up/Down, Async).
3. **03_Finite_State_Machines:** High-level system controllers including a **Traffic Light Controller**, **Vending Machine**, and **Car Parking logic** using Moore and Mealy models.
4. **04_FPGA_Implementation:** Physical constraint files (.xdc) for hardware targets like **Basys3** and **Nexys** boards.
5. **05_Simulation_Sources:** Custom testbenches developed to verify RTL functionality before hardware synthesis.
